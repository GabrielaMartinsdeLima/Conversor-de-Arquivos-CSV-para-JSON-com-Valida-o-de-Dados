# Conversor-de-Arquivos-CSV-para-JSON-com-Valida-o-de-Dados
Recebe um arquivo CSV enviado pelo usuário.  Converte esse CSV em JSON.  Valida o conteúdo de cada linha (ex: verifica se os campos obrigatórios estão presentes, formatos corretos de e-mail, datas, etc).  Salva o resultado em um arquivo .json.
